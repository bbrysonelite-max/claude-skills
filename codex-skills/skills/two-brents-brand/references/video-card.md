# The Video Card (every video follows this structure)

Production reality: Brent has two solid clone video loops + an ElevenLabs Professional Voice. AI Brent segments use the existing loops — write clone lines to be deliverable as voiceover-over-loop. Do not script clone visuals that require new footage.

## Structure

**1. HOOK (0:00–0:15)**
Problem or pattern interrupt. No intro, no logo, no "hey guys." Pulled from one of the four pillars. Cold open into the tension.
- Pillar 1 example: "In 1752 Franklin flew a kite. For forty years, electricity was a party trick. You're using AI as a party trick right now — and the window is closing."
- Pillar 2 example: "I lit six hundred thousand dollars on fire because I built before I asked. Let me save you the match."

**2. BANTER BRIDGE (0:15–0:30)**
The two Brents exchange 2-4 lines. This is where the hook's tension becomes the character dynamic. See voices.md for temperature.

**3. CLONE EDIFICATION (0:30–0:50)**
AI Brent delivers the credibility block (exact wording in SKILL.md), then hands off: "He's Brent, and he's going to show you how to stop staring at the lightning."
Film/produce ONCE as a reusable template segment with light variations. Do not rebuild per video — that's production procrastination.

**4. VALUE BLOCK (bulk of video)**
The actual teaching. One pillar, one idea, one demo or story. Show real screens, real businesses, real numbers where possible. Demos over theory. Brent Classic can interrupt with the audience's question; AI Brent or Human Brent answers it.

**5. SOFT MID-ROLL (at ~50–60% mark, AFTER a payoff moment)**
One line, five seconds: "If this is landing, subscribe — and the Academy link below is where we go deeper." Then straight back to content. Never at the front. Never begging.

**6. END CARD (final 20 seconds)**
Exactly ONE call to action. Pick per video: Subscribe / Probe Academy / Free Probe. Multiple CTAs kill conversion. Optionally close with the signature: AI Brent — "I'm just you with the current turned on." Point to next video if platform end-screen allows.

## Per-video checklist

- [ ] Maps to exactly one pillar
- [ ] Both Brents appear
- [ ] At least one banter beat
- [ ] Credibility block delivered by AI Brent (or template segment reused)
- [ ] One CTA only
- [ ] Passes "what do they need to know?" — a stranger gets it with zero background
- [ ] Clone lines deliverable over existing loops (no new clone footage required)
