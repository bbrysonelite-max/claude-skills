# Research Execution

## 9. Research execution

`needs-research` branches are executed by dispatched research agents using web search, public
APIs, and light scraping. Each returns a **draft answer with evidence attached** (source +
link/quote/number) for operator confirmation. No answer is `grounded` without evidence.

**Gating law for stages #2 and #3 (recorded here, enforced later):** no pipe, scraper, or API
feed gets built unless its refined ore matches a shape a **confirmed buyer will pay for.**
Free-API-first where possible; scrape only where you must; multiple pipes allowed — every one
justified by a buyer the Blueprint already proved.
